# Course Listing Website

Yeh ek simple course listing website hai jo React aur TailwindCSS ka use karke banayi gayi hai.

## Features
- **Home Page**: Search bar ke saath courses ki list. Har course card par image, naam aur price hai.
- **Details Page**: Click karne par simple details page khulta hai jisme sirf text aur 'Buy Now' button hai.
- **Mobile First**: Design mobile users ke liye optimized hai.

## File Structure
- `index.html`: Entry point for listing page.
- `course-details.html`: Entry point for details page.
- `data/courses.js`: Mock data for courses.
- `components/`: Reusable React components.
